pubDate: 2026-07-04

I have had my [old site](https://web.archive.org/web/20260704115945/https://fripe.dev/)
deployed for about two years and have grown increasingly unsatisfied with it.
During the summer, I finally decided to take the time to rewrite it into
what you see now.

Both the new and the old versions are built using [Astro](https://astro.build),
an excellent web framework for building content-focused static sites.
I have always had an itch for doing things at build-time, and Astro is built to
incentivize doing as little as possible on the client as the site is served.

I'm not yet entirely happy with the visual design of the site and think quite
a bit can be improved, and likely will be in the future, but its general
structure and layout feel satisfactory for now.
The main thing I am excited about is the blog I've added with this revision.
Hopefully the motivation will last long enough for me to put some actual content on it.
